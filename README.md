# mlpr
Sweet paddings with ./mlpr/mlpr.css

# Using

1. Copy to your `/style` (*or* `/css` *folder*)
2. `@import` to your main `style.css` file
3. Go to `html` file
4. Add `class` with padding
5. Enjoy sweet paddings

# Style

**ml** - margin-left
**mt** - margin-top
**mr** - margin-right
**mb** - margin-bottom

**pl** - padding-left
**pt** - padding-top
**pr** - padding-right
**pb** - padding-bottom

# Sample

`<h2 class='pl30 mt25'>`
is like:
`h2 {`
  `padding-left: 3vw;`
  `margin-top: 2vh;`
  `}`
